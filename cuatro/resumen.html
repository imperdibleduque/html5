La etiqueta <a> con el parámetro href se emplea para enlazar con otras páginas web y con cualquier recurso de Internet. El formato es siempre el mismo:

<a href="URL">contenido sobre el que se pulsará</a>

Las URL no tienen por qué estar siempre completas; podemos utilizar direcciones relativas, indicando sólo lo necesario para llegar a la página en cuestión. Esto lo emplearemos, al enlazar entre recursos dentro de un mismo sitio web.

Podemos también controlar en qué espacio se mostrará el contenido que se carga o realizar un enlace utilizando direcciones de correo electrónico.
